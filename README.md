# React Core Concepts

A structured repository built to revisit and reinforce React fundamentals from scratch, with hands-on examples of core concepts used in real-world applications.

## Why this repository?
Although I have already built full-stack MERN applications using React.js, I created this repository intentionally to revise and demonstrate a strong understanding of React fundamentals from the ground up.

This helps showcase:
- Strong conceptual clarity
- Clean component-based thinking
- Confidence with React basics before scaling into larger applications
- Interview-focused revision of core React concepts
